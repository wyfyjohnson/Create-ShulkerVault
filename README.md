## 📦 Overview

Welcome to **Shulker Vault**, a mod designed to create physically interactive, automation-friendly, quality-of-life storage blocks that feel perfectly natural alongside both vanilla Minecraft and the Create mod.

This isn't just an animated container. It's a hybrid storage solution combining the mobility of a Shulker Box with the utility of Create's Item Vaults and Toolboxes, topped off with modern inventory conveniences.

## ✨ Features

### The Base Shulker Vault
The core storage block of the mod, designed for everyday use and automation.
* **Vanilla Familiarity:** 27 inventory slots with physical lid-opening animations and player-lifting mechanics.
* **High Capacity:** Configurable per-slot stack multipliers (Defaults to `4x`, allowing massive storage in a small footprint).
* **Create Integration:** Fully supports Create-style interaction patterns. Works seamlessly with chutes, funnels, hoppers, and wrench pickups (which safely preserve your inventory!).
* **Quality of Life:** Built-in fast access and convenience interactions inspired by modern inventory mods.

### The Advanced Shulker Vault
An upgraded, logistics-focused tier for power users and complex Create factories. Crafted via Mechanical Crafters using a Shulker Vault, an Electron Tube, and a Packager.
* **All Base Features:** Upgrading your base vault preserves its entire inventory and metadata.
* **Direct Package Unpacking:** Acts as its own package destination. It can receive and internally unpack Create packages directly from Frogports, funnels, and hoppers without needing a separate Packager block.
* **Dedicated Package Slot:** Includes a specialized overflow slot to safely hold incoming packages if the main inventory cannot fully accommodate the unpacked items. 
* **Smart Memory:** Package-aware inventory memory behavior allows the block to predictably handle repeated deliveries of specific items.
* **Searchable GUI:** Easily find what you're looking for with built-in search capabilities.

## ⚙️ Requirements

* **Minecraft:** 1.21.1
* **Mod Loader:** NeoForge
* **Dependencies:** [Create Mod](https://modrinth.com/mod/create) 

*Note: Shulker Vault is currently developed exclusively for NeoForge. Fabric support is not currently planned.*

## 🛠️ Configuration & Compatibility
Shulker Vault is built to play nice with your favorite modpacks. 
* **Configurable Multipliers:** Adjust the per-slot stack multiplier in the configs to suit your modpack's balance.
* **Smart Overrides:** Built-in QoL interactions are designed to avoid conflicts with dedicated inventory mods (like Easy Shulker Boxes or Sophisticated Backpacks).
* **Stack Merging:** Items follow strict merging rules, ensuring components match perfectly before combining.
